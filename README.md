# Radar-Project

## Overview
This project uses Acconeer radar to detect objects and output distance data.

## Files

- 'speed_detector.py'
    Detects motion and speed of approaching and leaving targets.

- 'distance_radar.py'
    Detects distance of objects.

- 'obstacle_detector.py'
    Basic distance and speed detector combination

- 'bilateration_detector.py'
    Example script using two A121 radars (incomplete!)

- 'tracking_distance.py'
    Detects and tracks distance of objects. Determines if they are incoming or leaving depending on the tracking system. Determines motion using tracking logic. Estimates velocity of object using that logic. Warns user if object is getting too close. (incomplete!)
